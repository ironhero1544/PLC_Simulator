- **PLC RUN 버튼을 누른 후에도 래더 프로그램이 실행되지 않는 문제**
- Application의 `m_isPlcRunning` 상태와 ProgrammingMode의 `m_isMonitorMode` 상태가 제대로 연동되지 않음
- 시뮬레이션이 모니터 모드에서만 실행되는데, PLC RUN 상태가 모니터 모드를 자동으로 활성화하지 못함

### **🔧 최종 해결 방법**
```cpp
// [PPT: 새로운 내용 2 - PLC 상태 연동] PLC 실행 상태를 직접 전달하여 시뮬레이션을 제어합니다.
void ProgrammingMode::UpdateWithPlcState(bool isPlcRunning) {
    // PLC 상태에 따라 모니터 모드 자동 설정
    if (isPlcRunning) {
        m_isMonitorMode = true;
    }
    
    // 기본 업데이트 실행
    Update();
}
```

3. **src/Application.cpp**: Update() 루프에서 PLC 상태 전달 로직 추가

# **FX3U PLC 시뮬레이터: 실배선 모드**

## 1. 프로젝트 개요

본 프로젝트는 Mitsubishi FX3U PLC의 동작 원리를 학습할 수 있는 교육용 시뮬레이터입니다. 고가의 실습 장비나 물리적 공간의 제약 없이, 누구나 안전하고 직관적인 환경에서 PLC의 전기/공압 배선 및 회로 구성을 학습하는 것을 목표로 합니다.

- **주요 기술**: C++20, CMake, ImGui, GLFW, OpenGL
- **핵심 목표**: 반도체 설비 보전 기능사 수준의 PLC 교육 지원
- **최종 결과물**: 기능이 완성된 시뮬레이터 프로그램 및 발표 자료(PPT)

## 2. 해결하고자 하는 문제

기존 PLC 교육은 고가의 장비 구매 비용, 넓은 설치 공간 필요, 배선 오류로 인한 부품 손상 및 안전사고의 위험 등 여러 장벽이 존재합니다. 본 시뮬레이터는 이러한 문제들을 해결하여, 비용과 위험 부담 없이 반복적인 실습이 가능한 가상 학습 환경을 제공합니다.

## 3. 완성된 핵심 기능 (실배선 모드)

실배선 모드의 UI, 컴포넌트 시스템, 배선, 물리 시뮬레이션 기능이 완벽하게 설계 및 구현되었습니다.

### ⚙️ 3.1. 직관적인 사용자 인터페이스 (UI)

-   **메인 레이아웃**: 1440x1024 해상도의 창에 헤더, 툴바, 메인 작업 영역(캔버스 + 컴포넌트 목록), 상태바가 명확하게 구분되어 있습니다.
-   **커스텀 테마 및 폰트**: 가독성이 높은 밝은 테마와 한글을 지원하는 `unifont`를 적용하여 사용자 경험을 향상시켰습니다.
-   **동적 캔버스**: 마우스와 트랙패드를 모두 지원하는 정밀한 카메라 제어(줌, 팬)가 가능하며, 작업 편의를 위한 그리드와 도움말 툴팁을 제공합니다.

-   **타입 구분 배선**: `전기 배선`과 `공압 배선` 도구를 명확히 구분하여, 같은 타입의 포트끼리만 연결되도록 하여 사용자의 실수를 방지합니다.
-   **실시간 와이어 추적**: 컴포넌트를 이동하면 연결된 모든 배선이 실시간으로 따라 움직입니다.
-   **경로 편집**: 배선 중 캔버스 빈 공간을 클릭하여 중간점(Waypoint)을 추가하고, 복잡한 경로를 자유롭게 생성 및 수정할 수 있습니다.
-   **고급 스냅 기능**: 그리드, 포트, 수직/수평 가이드라인에 와이어가 자동으로 스냅되어 빠르고 깔끔한 배선이 가능합니다.

### 🚀 3.4. 현실적인 물리 시뮬레이션 엔진

-   **전기 시스템 (폐회로 모델)**:
    -   단순 전압 전파가 아닌, **`+24V`와 `0V`가 모두 연결된 완전한 폐회로**가 구성되어야만 부품(램프, 솔레노이드)이 작동하는 현실적인 전기 모델을 구현했습니다.
    -   스위치(버튼, 리미트 스위치) 및 센서의 상태에 따라 전기 회로의 연결 상태가 동적으로 변경됩니다.
    -   전압이 흐르는 배선은 색상(24V: 빨강, 0V: 파랑)이 변경되어 전류의 흐름을 직관적으로 확인할 수 있습니다.

-   **공압 시스템 (비선형 모델)**:
    -   **압력 조절**: FRL 유닛을 마우스 휠로 조작하여 시스템 전체에 공급되는 공압을 0~10 bar 사이로 정밀하게 조절할 수 있습니다.
    -   **압력 전파**: 매니폴드를 통한 압력 분배, 밸브를 통한 경로 변경 등 모든 공압 흐름을 BFS 알고리즘 기반으로 시뮬레이션합니다.
    -   **밸브 로직**: 전원 차단 시 초기 상태로 복귀하는 **단동 솔레노이드(Spring-return)**와, 상태를 기억하는 **복동 솔레노이드(Memory)**의 동작 차이를 완벽히 구현했습니다.

-   **기구학적 상호작용 (액추에이터 및 센서)**:
    -   **비선형 실린더 동작**: **최소 작동 압력(2.5 bar)** 개념을 도입하여, 압력 차가 기준치를 넘어야만 실린더가 움직입니다. 또한 힘은 압력 차의 **제곱에 비례**하여, 낮은 압력에서는 느리게, 높은 압력에서는 빠르게 가속하는 현실적인 움직임을 보여줍니다.
    -   **물리적 센서 감지**: 실린더 피스톤 로드의 **물리적 위치**와 센서 헤드의 **거리를 실시간으로 계산**하여, 전원 연결 여부와 상관없이 기구적으로 센서가 먼저 반응하도록 구현했습니다. 이 감지 신호가 전기 시스템의 스위치 역할을 수행합니다.

## 5. 교육적 가치 및 기술 하이라이트

본 프로젝트는 C++ 프로그래밍의 다양한 개념을 실제 문제 해결에 적용한 좋은 사례입니다.

-   **C++ 기초 활용**: `if-else`, `for`, `switch`, `std::vector`, `std::map` 등 기본 문법과 자료구조를 적극적으로 활용하여 프로그램의 핵심 로직을 구성했습니다.
-   **고급 C++ 기법**:
    -   **생성자 오버로딩/형변환 연산자**: `ImVec2`와 `Position` 간의 자동 타입 변환을 구현하여 코드의 가독성과 타입 안정성을 동시에 확보했습니다.
    -   **상태 저장 기계 (State Machine)**: 복동 솔레노이드 밸브의 '메모리' 기능을 구현하기 위해 컴포넌트의 내부 상태(`internalStates`)를 활용했습니다.
-   **라이브러리 활용**:
    -   **ImGui**: 복잡한 UI 및 컴포넌트 렌더링을 위해 `ImDrawList` API와 즉시 모드(Immediate Mode) UI 패러다임을 효과적으로 사용했습니다.
-   **알고리즘 응용**:
    -   **그래프 탐색 (BFS/DFS)**: 전기 및 공압 네트워크의 연결성을 분석하고, 전압/압력을 전파하는 핵심 알고리즘으로 사용되었습니다.
-   **물리 모델링**:
    -   **폐회로 전기 모델**: 실제 전기 회로의 동작 원리를 코드로 구현했습니다.
    -   **비선형 공압 모델**: 최소 작동 압력과 압력-힘의 제곱 관계를 적용하여 시뮬레이션의 현실성을 극대화했습니다.
